# Demo Materials

This directory contains demonstration materials for the data platform portfolio projects.

## Directory Structure

### screenshots/
Contains project screenshots and visual demonstrations:
- Dashboard views
- Application interfaces
- Data visualizations
- Feature highlights
- Before/after comparisons

**Recommended formats:** PNG, JPG, WebP
**Naming convention:** `project-name_feature-description_YYYYMMDD.png`

### videos/
Contains video demonstrations and walkthroughs:
- Feature demonstrations
- User workflow videos
- System architecture overviews
- Tutorial content
- Screen recordings

**Recommended formats:** MP4, WebM, GIF (for short clips)
**Naming convention:** `project-name_demo-description_YYYYMMDD.mp4`

## Guidelines

1. **Keep files organized** by project or feature
2. **Use descriptive names** that clearly indicate content
3. **Optimize file sizes** for web delivery where applicable
4. **Include captions** or brief descriptions in commit messages
5. **Update main project README** with links to demo materials

## Usage

Reference demo materials in project documentation using relative paths:
```markdown
![Feature Demo](./demos/screenshots/project-name_feature.png)
[Video Walkthrough](./demos/videos/project-demo.mp4)
```
